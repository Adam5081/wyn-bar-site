# WYN Social Bar — сайт

Лендинг лаундж-бара WYN (Астана, ул. Турар Рыскулов, 1) для GitHub Pages.

## Как редактировать контент

- **Акции и новости** — файл `js/data.js`, массив `NEWS`. Добавьте/удалите объект — карточка появится/исчезнет автоматически.
- **Меню и цены** — там же, объект `MENU`. ⚠️ Цены сейчас стоят примерные — замените на реальные.
- **Фото** — папка `img/`. Фото взяты из карточки заведения в 2ГИС; замените на свои в том же разрешении (имена файлов сохранить или поправить пути в `index.html`).
- **Контакты, часы** — прямо в `index.html` (секции `#info`, `#contacts`) и статус «открыто/закрыто» в `js/main.js` (константа `HOURS`).

## Пасхалка 🂡

На сайте спрятан покер (техасский холдем) против дилера:

- кликните масти в футере по порядку **♠ ♥ ♦ ♣**, или
- откройте сайт с якорем `#poker`.

Выигрыш даёт скидку: пара и ниже — 5%, две пары/сет — 10%, стрит и выше — 15%.
Промокод генерируется на экране — гость показывает его официанту.
Игра без ставок и денег. Логика — `js/poker.js` (реплики дилера — константа `SPEECH`, проценты — в функции `showdown`).
